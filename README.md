XBright
=======
Extremely simple and lightweight alternative to xbacklight that uses /sys/class/backlight/... (Linux 2.6+ ONLY)

Usage:
------
```bash
# xbright [+-=[0-123]]
```

Installation:
-------------
The program is auto-configured during make
```bash
make && sudo make install
```

Uninstallation:
---------------
```bash
sudo make uninstall
```
